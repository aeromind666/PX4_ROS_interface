# PX4_ROS_interface

>为方便算法开发者的控制PX4无人机的ROS库

## 📌 简介
 用于在ROS和PX4之间建立接口层级，便于算法开发者可以快速、安全的控制无人机。尤其便于在中科大高新校区北门开展实验的研究组。
